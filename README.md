# Практична робота з genericов у TypeScript

Цей репозиторій "dudai_pr_5" містить рішення практичного завдання, присвяченого використанню generic типів у TypeScript. Основна мета проєкту — продемонструвати роботу із загальними типами на прикладі базових функцій для інтернет‑магазину.

## Вміст

- **BaseProduct**, **Electronics**, **Clothing** — базові та специфічні типи товарів.
- Функції **findProduct** та **filterByPrice** використовують generics для пошуку та фільтрації товарів у масивах.
- **CartItem** — generic тип елемента кошика. Функції **addToCart** та **calculateTotal** забезпечують додавання товарів у кошик і підрахунок загальної вартості.
- Приклади використання функцій із тестовими даними для електроніки, одягу та книг.

## Структура

- `src/index.ts` — основний файл з усім кодом TypeScript.
- `tsconfig.json` містить базову конфігурацію компілятора (ES6, commonjs, strict режим).
- `package.json` визначає залежність `typescript` та скрипт `build`.
- `dist/index.js` — скомпільований JavaScript код.

## Використання

1. Клонувати репозиторій та встановити залежності:
   ```bash
   npm install
   ```
2. Скомпілювати TypeScript у JavaScript:
   ```bash
   npm run build
   ```
3. Запустити скомпільований код у Node.js, щоб побачити приклад роботи:
   ```bash
   node dist/index.js
   ```

Проєкт демонструє типобезпечну роботу з товарами різних категорій, застосовуючи generics для повторно використовуваних функцій і структур.
